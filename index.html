💬 팀 피드

· 팀 공동 사용: 화면 위 파란 배너의 [공유파일 연결]로 공유서버의 「팀업무보드_공유데이터.json」을 선택하면(최초 1회) 업무·일정·피드가 자동으로 팀 전체와 공유됩니다. 브라우저를 다시 열면 배너의 [허용하고 재연결] 클릭 한 번 — 허용 창에서 "모든 방문에서 허용"을 선택하면 다음부터 자동 연결됩니다. 제목 옆 상태가 초록색 "공유 연결됨"인지 확인하세요.
· 알람: 일정에 알람을 설정하면 지정 시각에 알림창과 소리로 알려줍니다. 🔔 알림 허용을 눌러두면 다른 창에서 일하는 중에도 Windows 알림으로 표시됩니다. (브라우저에 이 페이지가 열려 있어야 알람이 울립니다 — 아침에 한 번 열어두는 것을 권장)
· ★중요 일정은 페이지가 닫혀 있어도 텔레그램 자동 알림이 발송됩니다 (팀장 PC가 5분마다 공유 일정을 확인, 알람 미설정 시 기본 30분 전 발송).
· 업무 카드를 클릭하면 상세 내용 수정과 댓글 소통이 가능합니다. 왼쪽 색상: ■ 높음 ■ 보통 ■ 낮음
